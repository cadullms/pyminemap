# A map for Minecraft Pi Edition

## Getting started

* Before first run:

    ```sh
    pip install Flask
    ```

* Then start like this:

    ```sh
    FLASK_APP=pyminemap.py flask run --host=0.0.0.0
    ```
    And browse the site at http://localhost:5000
